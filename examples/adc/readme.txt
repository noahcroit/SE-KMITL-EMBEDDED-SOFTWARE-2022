Micropython example for raspberry pico with waveshare's pico-sensor-kit
Section - Analog-to-Digital Converter : ADC

List of the module
- LDR light sensor module ("LDR")
- Voltage Adjustable knob module ("VOL ADJ")
- Sound sensor module ("SOUND")

Module Connection Port
- LDR     -> ADC0 : GPIO26
- VOL ADJ -> ADC1 : GPIO27
- SOUND   -> ADC2 : GPIO28
          -> D
